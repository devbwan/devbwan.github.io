# 설정 가이드 인덱스

프로젝트의 모든 설정 가이드 문서를 한눈에 볼 수 있는 인덱스입니다.

## 📚 주요 설정 가이드

### 🔥 Firebase 설정

**`FIREBASE_COMPLETE_SETUP.md`** - Firebase 완전 설정 가이드
- Firebase 프로젝트 생성부터 코드 연동까지 모든 단계 포함
- 웹 앱 및 Android 앱 등록
- Firestore 데이터베이스 생성
- 보안 규칙 설정
- OAuth 도메인 설정
- 설정 검증 및 문제 해결

### 🔐 Google 인증 설정

**`GOOGLE_AUTH_COMPLETE_SETUP.md`** - Google 인증 완전 설정 가이드
- Firebase 프로젝트 설정
- Android 앱 등록
- SHA-1 인증서 지문 등록
- Firebase에서 Google 인증 활성화
- OAuth 동의 화면 설정 (상세 단계 포함)
- 프로젝트 설정 확인
- 앱 빌드 및 테스트
- 문제 해결 섹션

### 🌍 환경 변수 설정

**`ENV_COMPLETE_SETUP.md`** - 환경 변수 완전 설정 가이드
- 환경 변수 파일 구조
- 웹 및 Android 환경별 설정
- 필수 환경 변수 설정
- 환경 변수 사용 방법
- 플랫폼별 설정
- 문제 해결

## 📋 기타 설정 가이드

### 지도 및 위치 서비스
- **`GOOGLE_MAPS_SETUP_CHECKLIST.md`** - Google Maps 설정 체크리스트
- **`MAPS_TROUBLESHOOTING.md`** - 지도 관련 문제 해결

### 인증 서비스
- **`NAVER_AUTH_SETUP.md`** - 네이버 로그인 설정
- **`AUTH_SETUP.md`** - 인증 시스템 설정 (기본)

### 데이터베이스
- **`FIRESTORE_SETUP.md`** - Firestore 데이터베이스 설정
- **`FIRESTORE_ERROR_FIX.md`** - Firestore 오류 해결

### 배포
- **`GITHUB_PAGES_SETUP.md`** - GitHub Pages 배포 설정
- **`GITHUB_PAGES_DEPLOY.md`** - GitHub Pages 배포 가이드
- **`GITHUB_PAGES_CONFIG.md`** - GitHub Pages 설정
- **`GITHUB_PAGES_TROUBLESHOOTING.md`** - GitHub Pages 문제 해결

### 빌드 및 테스트
- **`APK_BUILD_GUIDE.md`** - APK 빌드 가이드
- **`ANDROID_BUILD_CHECKLIST.md`** - Android 빌드 체크리스트
- **`ANDROID_TEST_GUIDE.md`** - Android 테스트 가이드

## 🚀 빠른 시작

### 처음 설정하는 경우

1. **Firebase 설정**
   - `FIREBASE_COMPLETE_SETUP.md` 파일 열기
   - 단계별로 따라하기

2. **Google 인증 설정**
   - `GOOGLE_AUTH_COMPLETE_SETUP.md` 파일 열기
   - 단계별로 따라하기

3. **환경 변수 설정**
   - `ENV_COMPLETE_SETUP.md` 파일 열기
   - 웹 및 Android 환경별 설정

### 문제가 발생한 경우

1. 해당 기능의 통합 가이드에서 "문제 해결" 섹션 확인
2. 문제 해결 가이드 문서 확인
3. 체크리스트로 설정 확인

## ⚠️ 중요 사항

- 모든 설정은 **통합 가이드**에 포함되어 있습니다
- 중복된 가이드 파일들은 정리되었습니다
- 문제가 발생하면 통합 가이드의 "문제 해결" 섹션을 먼저 확인하세요

## 📝 가이드 문서 구조

```
설정 가이드/
├── FIREBASE_COMPLETE_SETUP.md      # Firebase 통합 가이드
├── GOOGLE_AUTH_COMPLETE_SETUP.md   # Google 인증 통합 가이드
├── ENV_COMPLETE_SETUP.md            # 환경 변수 통합 가이드
└── README_SETUP_GUIDES.md          # 이 파일 (인덱스)
```

---

**설정 중 문제가 발생하면 각 통합 가이드의 "문제 해결" 섹션을 참고하세요.**
